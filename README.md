# Task Manager App

A simple full-stack Task Manager application.

## Features

* User login
* Create tasks
* Delete tasks
* Real-time updates using Socket.io

## Tech Stack

**Frontend**

* React (Vite)
* TypeScript
* CSS

**Backend**

* Node.js
* Express
* Prisma
* PostgreSQL
* Socket.io

## Project Structure

```
backend/
frontend/
```

## How to Run Locally

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
